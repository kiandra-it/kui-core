# kui-core

## Install

```
bower install ng-kui-core --save
```

## Set up

Add module `ngKUICore` and `kguicore-partials`.

## Directives

- `kuiConfirm` `<button kui-confirm confirm-title="Test Title" confirm-prompt="Test Prompt" confirm-action="confirmed()">Click</button>`

## Services

- KConfirm
